## üß† AI Model: Graph Embeddings Simulation

The core of this application simulates the prediction step of a Graph Embeddings Model:

1.  **Model Training (Simulated):** The application uses pre-defined vectors (`INGREDIENT_VECTORS` in `app.py`). In a full project, these vectors would be the low-dimensional mathematical representations of ingredients generated by training a Graph Embedding Model on the **FlavorGraph** database.
2.  **Prediction Logic:** The `get_pairings` function calculates the **Cosine Similarity** between the input ingredient's vector and every other vector.
3.  **Output:** Ranks the results by the resulting similarity score, providing scientifically backed pairings. Ingredients that are mathematically "closer" in the vector space share more aromatic compounds, leading to surprising and excellent pairings.
